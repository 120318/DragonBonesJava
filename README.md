# DragonBones Java Runtime
这是java版本DragonBones Common Library，项目基于gradle，核心逻辑部分从DragonBones C++ Runtime 移植过来，
渲染方面使用了libGDX引擎编写，支持libgdx0.99以上的版本。

## 如何使用
1. 使用IDE导入build.gradle
2. 运行DesktopLauncher.java
